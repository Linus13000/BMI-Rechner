# BMI-Berechner
Dieses Programm dient zur Berechnung des BMI's einer Person.

# Benutzung: 
Programm herunterladen:
    wget https://github.com/Linus13000/BMI-Rechner/archive/master.zip
Programm auspacken:
    sudo apt install unzip
    unzip master
Programm starten:
    python BMI-Rechner-master/main.py
